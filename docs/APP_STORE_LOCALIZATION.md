| Field            | Max Length |
|------------------|------------|
| App Name         | 30 chars   |
| Subtitle         | 30 chars   |
| Promotional Text | 170 chars  |
| Description      | 4000 chars |
| Keywords         | 100 chars  |
| What's New       | 4000 chars |
